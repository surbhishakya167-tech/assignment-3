1. Game Idea
A balloon falls from the top of the screen.
The player controls a pin at the bottom.
The player shoots bullets to pop the balloon.
Score increases when balloon is popped.
Life decreases when balloon reaches the bottom.
 
 2. Rules
You have 3 lives.
Pop balloons to increase score.
If a balloon reaches the bottom → life -1.
Game ends when lives become 0.
Only one bullet can exist at a time.

 3. Controls
← (Left Arrow) → Move left
→ (Right Arrow) → Move right
SPACE → Shoot bullet

 4. Features
Moving balloon (falling)
Moving player pin
Shooting bullets
Score system
Life system
Random balloon positions
Smooth screen refresh
